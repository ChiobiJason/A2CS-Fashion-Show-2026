# A2CS Fashion Show 2026: The Future of Style

A premium, high-end digital experience for the **A2CS Fashion Show 2026**. This platform showcases the convergence of innovation, cultural heritage, and futuristic design.

## ✨ Features

- **Cinematic Experience**: A dramatic, interactive Hero section with animated blob visuals and grid overlays.
- **Modern Aesthetic**: A "Rich Black" and "Electric Green" color palette using premium typography (**Outfit** & **Inter**).
- **Advanced Glassmorphism**: High-end UI components with sophisticated blur effects and subtle interactive states.
- **Curated Sections**:
  - **The Vision**: An overview of the show's mission.
  - **The Collection**: Interactive category exploration for different fashion disciplines.
  - **The Curators**: A showcase of the expert judges with interactive portrait cards.
  - **Registration Hub**: Streamlined sign-up forms for models and volunteers.
- **Dynamic Interactions**: Smooth scroll-reveal animations and high-performance hover effects.

## 🛠️ Tech Stack

- **Vue 3**: Composition API for a scalable and reactive frontend.
- **Vite**: Ultra-fast build tool and development server.
- **Vanilla CSS**: Custom design system built for maximum performance and visual precision.
- **Intersection Observer**: For high-performance scroll animations.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16.x or later)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Configure environment variables:
   Create a `.env` file in the root with:
   ```env
   VITE_APPS_SCRIPT_URL=your_google_apps_script_url
   ```

### Development

Run the development server:
```sh
npm run dev
```

### Production

Build the production-ready bundle:
```sh
npm run build
```
